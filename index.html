🟢 ЗЕЛЁНЫЙ КОД — Базовый уровень

Все особые положения аннулированы. У командования нет сведений о наличии угрозы.

Регламент СБ

  • Основное вооружение скрыто
  • Разрешена выдача станнеров
  • Огнестрельное только с нелетальными
  • Задержание при непосредственном наблюдении
  • Требуется санкция для задержаний
  • Запрещены обыски без письменного разрешения

Регламент экипажа

  • Свободное перемещение по допуску
  • Рекомендовано: режим "Координаты"
  • Запрещено: разглашение конфиденциальной информации
  • ЗОДы закрыты при обоснованной причине
  • Хранилище открывается при необходимости
🔵 СИНИЙ КОД — Уровень средней угрозы

Командование получило данные о враждебной активности. Положение упрощает процедуры.

Регламент СБ

  • Разрешено нелетальное вооружение
  • Летальное — с санкции ГСБ
  • Открытое ношение оружия допускается
  • Санкция не требуется при основаниях
  • Усиленный контроль доступа
  • Проверка всех входящих/выходящих
  • Усиленный режим охраны критических систем

Регламент экипажа

  • Ограничение свободного перемещения
  • Соблюдение уровней допуска
  • Обязательная идентификация на КПП
  • Следовать указаниям СБ
  • Включить режим "Координаты"
  • Избегать скоплений экипажа
  • ЗОДы закрываются на контроль
🟡 ЯНТАРНЫЙ КОД — Повышенный уровень

Значительные враждебные угрозы. Сотрудники СБ следуют процедурам усиления.

Регламент СБ

  • Разрешено летальное вооружение из арсенала
  • Ограничения на ношение оружия сняты
  • Обязательное ношение защитного снаряжения
  • Право на проведение специальных операций
  • Усиленный контроль всех зон
  • Обыски без санкции
  • Полная проверка всех помещений

Регламент экипажа

  • Строгий контроль перемещений
  • Обязательная идентификация на КПП
  • Запрет нахождения в запретных зонах
  • Следовать указаниям СБ
  • Включить режим "Координаты"
  • Избегать скопления экипажа
  • Готовность к эвакуации
🔴 КРАСНЫЙ КОД — Максимальная угроза

Начало военной операции. Применение летальной силы. Любой сотрудник обязан подчиняться приказам СБ.

Условие: Присутствие 3+ угроз Янтарного уровня или угроза граничащая с полным уничтожением.

Регламент СБ

  • Любое вооружение из арсенала
  • Обязательное ношение полного комплекта защиты
  • Сила без предупреждения
  • Специальные операции в приоритетном порядке
  • Проникновение в любые зоны
  • Любые средства подавления
  • Полный локдаун всех зон
  • Блокировка всех выходов и входов

Регламент экипажа

  • Полный запрет на свободное перемещение
  • Обязательная эвакуация в безопасные зоны
  • Строгий контроль всех перемещений
  • Немедленное подчинение приказам СБ
  • Соблюдение полной тишины
  • Готовность к экстренной эвакуации
  • Полная эвакуация некритичного персонала
  • Блокировка всех внешних коммуникаций

Особые положения

  • Приоритет всех ресурсов на устранение угрозы
  • Все решения принимаются только командованием
  • Отказ от приказов карается по максимально строгому регламенту
⚡ ГАММА КОД — Особый режим максимальной мобилизации

Станция под серьёзной угрозой. ЦК проводит значительное усиление СБ. Неограниченные превентивные меры.

Важно: Только по решению ЦК при крупных угрозах и невозможности эвакуации.

Регламент СБ

  • Любое вооружение на станции
  • Полный комплект защиты
  • Любые спецсредства
  • Мобилизация всех сил СБ
  • Полная изоляция критических зон
  • Проникновение в любые помещения
  • Приоритетное использование ресурсов

Регламент экипажа

  • Строгий контроль всех перемещений
  • Эвакуация в безопасные зоны
  • Запрет нахождения в запретных зонах
  • Полное подчинение приказам СБ
  • Отключение личных коммуникаторов
  • Соблюдение режима тишины
  • Готовность к эвакуации

Особые положения

  • Содержание под стражей допускается без процессуальных формальностей
🟣 ФИОЛЕТОВЫЙ КОД — Биологическая/химическая угроза

Угроза локализованной эпидемической вспышки. Объявлен карантин. Персонал выполняет инструкции медотдела.

Регламент СБ

  • Обязательные средства химзащиты
  • Выдача средств защиты и костюмов
  • Разрешено летальное вооружение
  • Организация карантинных зон
  • Изоляция заражённых участков
  • Контроль распространения заражения

Регламент экипажа

  • Немедленное надевание средств защиты
  • Соблюдение карантинного режима
  • Выполнение указаний СБ
  • Запрещено: снятие средств защиты
  • Запрещено: контакт с заражёнными зонами
  • Полная изоляция заражённых зон
🟠 ЖЁЛТЫЙ КОД — Повреждения станции

Серьезная угроза для технического функционирования объекта. Персонал выполняет инструкции инженерного персонала.

Регламент СБ

  • Разрешено нелетальное вооружение
  • Обязательное ношение защитного снаряжения
  • Усиленный контроль критических зон
  • Организация дополнительных постов наблюдения
  • Профилактические проверки
  • Мониторинг коммуникаций

Регламент экипажа

  • Ограничение доступа в некоторые зоны
  • Обязательная идентификация при проходе
  • Поддержание повышенной бдительности
  • Информирование СБ о подозрительных действиях
  • Готовность к повышению уровня угрозы
  • Включение "Маячка Слежения"

🎖️ Глава СБ

• Следить за исполнением КЗ и НРП

• Оповещать об угрозах, устанавливать коды

• Командовать отделом

• Пинпоинт НЯД

• Имплант защиты разума

• Выдаёт разрешения на оружие

🔒 Смотритель

• Учёт вооружения в арсенале

• Поддержание чистоты брига

• Надзор за заключёнными

• Обеспечение офицеров боеприпасами

• Командует при отсутствии ГСБ

🛡️ Офицер

• Выполнение приказов ГСБ/Смотрителя

• Идеальное знание КЗ

• Аресты до 3XX статей включительно

• До 4 единиц оружия

🪖 Кадет

• Только с напарником!

• Знание КЗ и НРП

• Стандартная экипировка

• Нет летального оружия

• Нет экзо-костюмов
