## 🚀 Развёртывание на GitHub Pages

1. **Создайте репозиторий на GitHub**
   - Перейдите на [github.com](https://github.com)
   - Нажмите "New repository"
   - Назовите репозиторий (например, `ss14-wiki`)

2. **Загрузите файлы**
   - Клонируйте репозиторий на компьютер
   - Скопируйте все файлы из папки `ss14-wiki` в корень репозитория
   - Запушьте изменения:
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

3. **Включите GitHub Pages**
   - Перейдите в Settings → Pages
   - В Source выберите "Deploy from a branch"
   - Выберите branch "main" и папку "/ (root)"
   - Нажмите Save

4. **Дождитесь развёртывания**
   - GitHub автоматически развернёт сайт
   - Через 1-2 минуты сайт будет доступен по адресу: `https://[username].github.io/[repo-name]/`

## 📚 Содержание

### НРП Кодов
Система уровней угрозы станции:
- 🟢 Зелёный код — базовый уровень безопасности
- 🔵 Синий код — уровень средней угрозы
- 🟡 Янтарный код — повышенный уровень угрозы
- 🔴 Красный код — максимальная угроза
- ⚡ Гамма код — особый режим максимальной мобилизации
- 🟣 Фиолетовый код — биологическая/химическая угроза
- 🟠 Жёлтый код — повреждения станции

### НРП Службы Безопасности
Должностные обязанности и права:
- Глава Службы Безопасности (ГСБ)
- Смотритель
- Офицер СБ
- Кадет СБ
- Бригмедик
- Детектив

### Космический Закон
Уголовный кодекс НаноТрейзен:
- Статьи 1XX-5XX
- Модификаторы (оправдательные и отягощающие)
- Враги корпорации
- Процедурные регламенты
- Классификация газов

## 🎨 Кастомизация

Цвета определены в CSS переменных:
```css
--accent-green: #00ff88;   /* Основной акцент */
--accent-blue: #3498db;    /* Синий код */
--accent-amber: #f1c40f;   /* Янтарный код */
--accent-red: #e74c3c;     /* Красный код */
--accent-violet: #9b59b6;  /* Фиолетовый код */
```
