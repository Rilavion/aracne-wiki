## 🚀 Развёртывание на GitHub Pages

### 1. Создайте репозиторий
```bash
# На GitHub создайте новый репозиторий (например, ss14-wiki)
```

### 2. Загрузите файлы
```bash
# Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/ss14-wiki.git
cd ss14-wiki

# Скопируйте все файлы проекта в корень репозитория
# (index.html, codes.html, security.html, law.html, styles.css, script.js, README.md)

# Коммитите и пушьте
git add .
git commit -m "Add SS14 Wiki"
git push origin main
```

### 3. Включите GitHub Pages
1. Перейдите в **Settings** → **Pages**
2. Source: **Deploy from a branch** → **main** → **/ (root)**
3. Нажмите **Save**

### 4. Готово! 🎉
Через 1-2 минуты сайт будет доступен:
```
https://YOUR_USERNAME.github.io/ss14-wiki/
```

### 👔 Должности СБ
- **Глава Службы Безопасности (ГСБ)** — командование отделом
- **Смотритель** — контроль брига и арсенала
- **Офицер СБ** — патрулирование и аресты
- **Кадет** — обучение под руководством наставника
- **Бригмедик** — медицинская помощь в бриге
- **Детектив** — расследования и криминалистика

### ⚖️ Космический Закон
- **1XX** — Ущерб станции / Неподобающее поведение
- **2XX** — Саботаж / Драки / Кража имущества отдела
- **3XX** — Серьёзный саботаж / Увечья / Контрабанда
- **4XX** — Грандиозный саботаж / Тяжкие увечья / Мятеж
- **5XX** — Вопиющий саботаж / Убийство / Критические преступления

### 🛡️ Модификаторы
**Оправдательные:**
- П-1: Сотрудничество (−50%)
- П-2: Отсутствие умысла (−1 уровень)

**Отягощающие:**
- О-1: Рецидив (+10 мин)
- О-2: Должностное (+15 мин)
- О-3: Против должностных (+50%)
- О-4: Помеха расследованию (+50%)
- О-5: Сопротивление аресту (+50%)
- О-6: Критический рецидив (+100%)
- О-7: Пособничество (+90%)

## 📝 Добавление контента

### Новые страницы
1. Скопируйте структуру существующей страницы
2. Обновите навигацию во всех HTML файлах
3. Добавьте ссылку в главное меню

### Новые статьи КЗ
Добавьте блок `<div class="article">` в соответствующую секцию XX*:
```html
<div class="article">
    <div class="article-header">
        <span class="article-code">XXX</span>
        <h4>Название статьи</h4>
        <span class="article-level">XX0</span>
    </div>
    <p><em>Описание...</em></p>
    <ul>
        <li>Пункт 1</li>
        <li>Пункт 2</li>
    </ul>
</div>
```

### Новые коды тревоги
Добавьте блок `<section class="code-section">` в codes.html:
```html
<section class="code-section" id="code-id">
    <div class="code-header code-color-header">
        <span class="code-indicator">🔮</span>
        <h3>Название кода</h3>
        <span class="code-badge">Уровень</span>
    </div>
    <!-- Остальное содержание -->
</section>
```
