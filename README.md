# SS14 Quick Reference

Минималистичный справочник для быстрой ориентации прямо во время игры.

## 📁 Структура

```
ss14-wiki/
├── index.html    # Всё в одном файле
├── styles.css    # Стили
├── script.js     # Логика
└── README.md
```

## ✨ Возможности

- **🎨 Раскрашенные таблицы** — переключатель вкл/выкл
- **📢 Коды тревоги** — быстрый обзор всех уровней
- **⚠️ Угрозы** — уровни опасности антагонистов
- **⚖️ Статьи КЗ** — сводная таблица 1XX-5XX
- **📋 Модификаторы** — оправдательные и отягощающие
- **👔 Должности СБ** — краткий справочник
- **🧮 Калькулятор** — расчёт срока заключения

## 🚀 Установка

1. Создайте репозиторий на GitHub
2. Загрузите 4 файла в корень
3. Включите GitHub Pages: Settings → Pages → Deploy from branch → main
4. Готово!

## 🎮 Использование

1. Откройте сайт на втором мониторе или втором устройстве
2. Используйте боковое меню для навигации
3. Включите/выключите цвета таблиц по желанию
4. Используйте калькулятор для быстрого расчёта приговоров

## ⚙️ Настройки

- **Цвета таблиц** — сохраняются в localStorage
- **Сохранение** — переключатель работает между сессиями
